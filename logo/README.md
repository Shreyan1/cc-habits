# cc-habits logo assets

## Files

| File | Usage |
|---|---|
| `cc-habits-icon.svg` | Mark only · transparent bg · dark ink |
| `cc-habits-icon-lime.svg` | Mark on acid-lime tile · dark ink |
| `cc-habits-icon-dark.svg` | Mark on deep-green tile · lime ink |
| `favicon.svg` | Favicon · acid-lime bg · auto-scales |
| `cc-habits-lockup.svg` | Horizontal lockup · dark ink |
| `cc-habits-lockup-dark.svg` | Horizontal lockup · lime on dark |
| `cch-lockup.svg` | Compact lockup · dark ink |
| `cch-lockup-dark.svg` | Compact lockup · lime on dark |
| `icon-512.png` | App icon 512×512 (lime tile) |
| `icon-192.png` | PWA icon 192×192 |
| `icon-180.png` | Apple touch icon 180×180 |
| `icon-32.png` | Favicon PNG 32×32 |
| `icon-16.png` | Favicon PNG 16×16 |

## Colors
- Acid-lime: `#B5FF1C`
- Deep-green: `#102600`
- Cursor-blue: `#00B7FF`

## Typography
- Wordmark: Silkscreen (Google Fonts, weight 400)
- Body: JetBrains Mono

## HTML usage
```html
<link rel="icon" type="image/svg+xml" href="/assets/logo/favicon.svg">
<link rel="icon" type="image/png" sizes="32x32" href="/assets/logo/icon-32.png">
<link rel="apple-touch-icon" href="/assets/logo/icon-180.png">
```
